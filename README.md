# 국제건강관리지도사협회 홈페이지

이 프로젝트는 Vercel에 바로 배포할 수 있도록 구성된 정적 홈페이지입니다.

## 포함된 내용
- 협회 브랜드 홈 소개
- 교육 프로그램 소개
- 인증 시스템 흐름 안내
- 회원 구성 소개
- 문의 연락처 섹션

## GitHub / Vercel 배포 방법
1. 이 폴더를 GitHub 저장소로 업로드합니다.
2. GitHub 저장소에서 Vercel 프로젝트를 새로 연결합니다.
3. 배포 대상 폴더는 루트 폴더로 설정합니다.
4. 기본 설정으로 배포하면 정적 사이트가 올라갑니다.

## 주의사항
- Vercel 계정은 기존 건강이음/iahi 계정과 분리된 상태로 확인하는 것이 안전합니다.
- Google 계정 로그인 정보도 함께 확인해 주세요.
- 실제 회원가입, 결제, 교육영상, 문제은행, 자격증 PDF 발급 기능은 이후 백엔드 연동이 필요합니다.
- 배포 후에는 실제 협회 정보, 교육과정명, 자격증 명칭으로 문구를 수정하는 것이 좋습니다.
