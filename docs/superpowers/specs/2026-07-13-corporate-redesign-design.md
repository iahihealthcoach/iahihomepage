# 홈페이지 세련된 기업 이미지 리디자인 — 설계 문서

## 배경
국제건강관리지도사협회 홈페이지(index.html / styles.css / script.js)는 현재 파스텔 톤의 티얼-블루 그라디언트, SVG 일러스트 중심의 일반적인 스타트업풍 디자인이다. 사용자는 세 가지 참고 이미지(서울이스탠다드 — 딥네이비+실사진, DotCle — 블랙+글로시 3D+미니멀 타이포, Cafe24 — 비비드 블루+인물사진)를 제시하며 "전문교육기관처럼 세련된 기업 이미지"로 전면 재단장을 요청했다.

## 목표
- 협회/교육기관다운 신뢰감과 프리미엄 톤을 갖춘 비주얼로 전환한다.
- 기존 8개 섹션(히어로·소개·운영흐름·교육상품·강점·사례·인증·회원·문의)과 콘텐츠 구조, 앵커 링크는 그대로 유지한다.
- 콘텐츠(카피)는 변경하지 않고 색상·타이포·이미지·여백·카드 스타일만 재단장한다.

## 채택 방향: "Deep Navy Editorial"
세 참고안 중 서울이스탠다드(딥네이비+실사진)를 기본 톤으로 하고, DotCle의 정제된 타이포/여백 감각을 더한다. Cafe24의 비비드 블루+인물사진 톤은 채택하지 않는다(커머스 느낌이 강해 협회 정체성과 맞지 않음).

### 컬러 시스템
- 다크 네이비 배경(그라디언트): 히어로 섹션 + 문의(CTA) 섹션에만 사용
  - `#0a1929` → `#0d2c47` → `#123a5c`
- 나머지 섹션: 화이트(`#ffffff`) / 라이트그레이(`#f6f8fb`) 교차 배경 유지 (현재 `.section-alt` 패턴 유지)
- 포인트 컬러(신규): 골드/앰버 `#c9a227` — 뱃지, 아이콘 강조, 구분선, hover 상태 등 "인증/전문성" 뉘앙스에 사용. 기존 블루(`--primary`, `--accent`)는 링크·버튼 등 인터랙션 요소에 유지
- 텍스트: 다크 네이비(`--text`) 유지, 히어로 내부는 화이트/화이트 70% 투명도

### 타이포그래피
- 폰트는 Noto Sans KR 유지(신규 폰트 로드 안 함)
- 헤드라인(h1, 섹션 h2)은 font-weight 900 + letter-spacing 축소 + 사이즈 확대(clamp 값 상향)로 임팩트 강화
- 본문/설명 텍스트는 현재 유지

### 히어로 섹션
- 다크 네이비 그라디언트 배경, 좌측 텍스트 / 우측 실사진(Unsplash) 2단 구성 유지 (현재 grid 구조 재사용)
- 현재 `hero-image.svg` 일러스트 카드를 제거하고, 실제 스톡 사진(교육/컨퍼런스 장면)으로 교체. 사진 위에 다크 네이비 듀오톤 오버레이를 얹어 톤을 통일
- "핵심 운영 영역" 리스트는 사진 아래 화이트 카드 형태로 유지

### 카드 및 섹션 스타일
- 카드(`.card`, `.feature-card`, `.pill-item`, `.case-card`)는 화이트 서피스 유지, `border-radius`를 축소(1.2rem → 0.6~0.75rem)해 더 각지고 프리미엄한 느낌 부여
- 카드 그림자는 더 낮고 은은하게 조정
- 뱃지/아이콘류에 골드 포인트 컬러 적용 (`.eyebrow`, `.product-badge` 등)
- "사례"(`#cases`) 섹션 카드에 스톡 사진 썸네일을 상단에 추가 (현재는 텍스트만 존재)
- 섹션 간 여백(`.section` padding)을 소폭 확대해 여유 있는 톤 확보

### 이미지 소싱
- 무료 스톡 사진을 Unsplash CDN(`images.unsplash.com`)에서 직접 hotlink로 사용 (다운로드/저장 없이 `<img src>` 직접 참조)
- 필요 장면: 히어로용 1장(회의/컨퍼런스), 사례 카드용 3장(교육 현장, 전문가 회의, 발표/시상)
- 구현 단계에서 실제 이미지 URL의 로딩 가능 여부를 확인한 뒤 확정한다
- 기존 `hero-image.svg`는 더 이상 참조하지 않지만 파일은 삭제하지 않는다(다른 곳에서 참조 여부 확인 후 정리는 구현 단계 재량)

### 로고/네비게이션
- 현재 `logo.svg` + 브랜드명 구성은 유지
- 헤더 배경은 화이트 유지, sticky 동작 변경 없음

### 반응형
- 기존 브레이크포인트(860px, 680px) 구조 유지, 그리드 컬럼 수 변경 없음
- 히어로 실사진은 모바일에서 텍스트 위/아래 배치로 자연스럽게 스택 (`grid-template-columns: 1fr`)

## 범위 밖 (Out of scope)
- 섹션 추가/삭제, 콘텐츠(카피) 수정
- 신규 폰트 로드
- 실제 협회 촬영 사진으로의 교체(추후 사용자가 실사진 확보 시 별도 작업)
- 다국어/CMS 연동 등 기능적 변경

## 구현 파일
- `styles.css`: 컬러 변수, 히어로, 카드, 타이포 스타일 전면 수정
- `index.html`: 히어로 이미지 마크업을 `hero-image.svg` → 스톡 사진 URL로 교체, 사례 카드에 이미지 요소 추가
- `script.js`: 사례 카드 렌더링 시 이미지 필드 추가 (필요 시)

## 검증 방법
- 로컬에서 정적 파일을 브라우저로 열어 데스크톱/모바일(680px, 860px 기준) 레이아웃 확인
- 히어로/카드 이미지가 정상 로드되는지 확인
- 기존 앵커 링크(`#about`, `#program`, `#cases` 등) 동작 확인
